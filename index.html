USER EXPERIENCE (UX) ANALYSIS FOR FLEXTRADE SHOPPING APP

In this UX analysis for FlexTrade, we used Microsoft Excel to assess app performance from 2020–2023. Key metrics included bounce rate (49%), conversion rate (50%), and add-to-cart rate (47–52%). Findings showed that longer user sessions and higher cart additions led to better conversions. Issues like app crashes and poor checkout flow caused cart abandonment and high bounce rates. Dashboards revealed declining user engagement over time, signaling the need for intervention. Recommendations focused on UX improvements, personalised suggestions, and awareness campaigns.

Enhancing User Experience (UX): A Tableau A/B Test analysis

This project leveraged Tableau to analyse A/B test data for DigitalPath Innovations to uncover which webpage version drove better user engagement and conversions. Version A outperformed with users spending an average of 5 minutes on-site and viewing 10 pages, compared to 3.3 minutes and 5 pages for Version B. Despite similar checkout initiation rates (~62%), Version A achieved a remarkable 70.5% conversion rate, nearly double Version B's 38.2%. The analysis also highlighted Product 3 as the most viewed and revealed low feedback submission (~18%), pointing to clear areas for strategic improvement. This project showcased the power of data visualisation in decision-making, turning raw metrics into actionable insights.

Techtronix-Innovation Project

Techtronix Innovation project dashboard presents a comprehensive analysis of the techtronicx innovation's 2023 transaction data with PowerBI solutions, covering sales, customer behavior, product performance, and financial metrics. Microchips were the top revenue earners, while the Industrial sector led in profitability. Despite high transaction volumes, some products and regions delivered low margins. The Asia-Pacific region generated the highest profit, while North America underperformed. Inventory mismatches and seasonal demand variations were identified. Recommendations focused on optimizing high-margin products, improving regional strategies, and retaining key customers. Overall, the project provided actionable insights to drive efficiency and growth.
